https://parallax.kr/en/docs/api 페이지에 대한 분석 결과입니다. 이 페이지는 개발자들을 위한 **"OpenAI 호환 API 문서"**로, 깔끔한 인터페이스와 실용적인 코드 예제를 제공합니다.

1. 문서 구조 및 레이아웃
3-컬럼 구조: 왼쪽에는 카테고리별 사이드바, 중앙에는 엔드포인트 설명, 오른쪽(또는 본문 하단)에는 코드 예제가 배치된 표준적인 API 문서 레이아웃을 따릅니다.
카테고리 구성: Getting Started, Chat, Images, Video, Audio, Storage, Workflows 등 기능별로 체계적으로 분류되어 있습니다.
2. 디자인 및 사용자 경험 (UX)
브랜드 일관성: 메인 사이트의 프리미엄한 글래스모피즘(Glassmorphism) 스타일과 타이포그래피를 유지하면서도, 정보 전달의 명확성을 위해 배경을 밝게 처리했습니다. (다크 모드 지원 포함)
코드 편의성: 모든 코드 블록에 bash/curl 예제가 포함되어 있으며, 클릭 한 번으로 복사할 수 있는 기능을 제공합니다.
OpenAI 호환성: /v1/chat/completions 엔드포인트를 제공하여, 기존 OpenAI 라이브러리를 사용하는 개발자들이 쉽게 마이그레이션할 수 있도록 설계되었습니다.
3. 독특한 기능: Workflows API
Parallax AI만의 차별점인 'Iris' 데스크톱 앱에서 만든 자동화 워크플로우를 API를 통해 원격으로 실행할 수 있는 Workflows 엔드포인트가 상세히 설명되어 있습니다. 이는 단순한 챗봇을 넘어 'AI 에이전트 실행 플랫폼'으로서의 성격을 잘 보여줍니다.

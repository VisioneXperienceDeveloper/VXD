# Blog Design System: Elegant Paper

## Core Vision
A reading-focused experience that feels like a premium digital magazine. "Elegant Paper" prioritizes white space, refined typography, and a calm, sophisticated atmosphere.

## 1. Color Palette (Sophisticated Neutrals)
- **Primary**: `#111827` (Slate 900) - Deep text and strong accents.
- **Background**: `#ffffff` (Light) / `#0a0a0a` (Dark).
- **Secondary**: `#4b5563` (Gray 600) - Meta info and secondary text.
- **Accent**: `#c084fc` (Purple 400) - Category tags and link underlines.
- **Muted**: `#f3f4f6` (Gray 100) - Blockquotes and code backgrounds.

## 2. Typography
- **Headings**: `Playfair Display` or `Outfit`, Extra Bold (800).
- **Body**: `Inter`, Light (300) for large text, Regular (400) for standard.
- **Serif Option**: For long-form content, use a high-quality serif for maximum readability.

## 3. UI Components
- **Article Cards**:
  - Minimal borders, large featured images with rounded corners (`24px`).
  - Hover: Subtle image zoom and shadow transition.
- **Navigation**:
  - Fixed top bar with a clean, centered logo.
- **Typography-centric**: 
  - Generous line-height (`1.7`) and paragraph spacing.

## 4. Visual Effects & Animations
- **Fade-ins**: Content should reveal smoothly as the user scrolls.
- **Transitions**: Ease-in-out for link states and button hovers.

## 5. Implementation Rules
- Max content width: `720px` for optimal reading speed.
- Focus on accessibility: High contrast ratios for all text.
- Grid system: Clean `12-column` grid with large gutters (`32px`).

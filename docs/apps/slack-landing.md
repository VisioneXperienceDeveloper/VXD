# Slack Landing Page

Marketing and landing page for the Slack clone.

## Overview

- **Framework**: Astro
- **Directory**: `apps/slack-landing`
- **Port**: 5101 (Local)
- **Production URL**: `https://slack-vxd.vercel.app` (Example)

## Key Features

- **Static Performance**: Minimal JavaScript for fast load times.
- **Marketing Sections**: Hero, features, pricing, and FAQ.
- **Responsive Design**: Optimized for all screen sizes.

## Technology Details

- **Tailwind CSS v4**: Consistent styling with the rest of the monorepo.
- **Astro Components**: Highly efficient static rendering.

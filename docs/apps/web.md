# Web Application

The main portfolio and legacy blog application.

## Overview

- **Framework**: Next.js 15 (App Router)
- **Directory**: `apps/web`
- **Port**: 3000 (Local)
- **Production URL**: `https://visionexperiencedeveloper.com`

## Key Features

- **Portfolio**: Showcase of projects and resume.
- **Legacy Blog**: Integrated blog functionality (moving to `blog-web`).
- **CMS**: Integration with Notion API.
- **i18n**: Multi-language support using `next-intl`.

## Technology Details

- **Styling**: Tailwind CSS v4
- **UI Components**: Radix UI, `@vxd/ui` package
- **Forms**: React Hook Form, Zod
- **Backend**: Notion API, Supabase (for some metadata)

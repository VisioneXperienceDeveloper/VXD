# Slack Web Application

A full-featured web-based Slack clone.

## Overview

- **Framework**: Next.js 15 (App Router)
- **Directory**: `apps/slack-web`
- **Port**: 5103 (Local)
- **Production URL**: `https://slack.visionexperience.com` (Example)

## Key Features

- **Real-time Messaging**: Instant message delivery and updates.
- **Workspaces & Channels**: Multi-tenant structure with channel management.
- **Threads & Reactions**: Rich communication features.
- **File Uploads**: Image and file attachments.

## Technology Details

- **Backend**: Convex (`@vxd/slack-backend`)
- **Authentication**: Clerk (`@vxd/auth`)
- **UI**: Shadcn UI, Lucide Icons
